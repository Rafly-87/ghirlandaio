| 1. | Efektivitas penerapan sistem Library Automation Project (LAP) pada perpustakaan pengguna | Bambang Suryo Putro | 2010 | Skripsi ini membahas Aplikasi database perpustakaan Library Automation Project (LAP) dan penerapannya dalam pengelolaan perpustakaan. Aplikasi ini dapat membantu dalam pengelolaan perpustakaan seperti pengolahan data, peminjaman dan pengembalian koleksi (sirkulasi), administrasi anggota dan penelusuran koleksi. Melalui penelitian, didapatkan bahwa para pustakawan merasa terbantu dengan aplikasi ini, akan tetapi terdapat beberapa kekurangan seperti aplikasi tersebut belum terhubung dengan internet sehingga sulit untuk mengembangkan layanan perpustakaan dan kurangnya pengetahuan sdm perpustakaan.Saran yang dapat diberikan atas permasalahan di skripsi tersebut adalah bagi pengelola perpustakaan untuk mencoba aplikasi lain serta meningkatkan pengetahuan teknologi sehingga tidak gagap dalam penggunaan aplikasi. Sementara itu, untuk pengembang aplikasi LAP untuk mengembangkan aplikasi sehingga dapat dipakai mengikuti perkembangan zaman. | 
| 2. | Analisis Penerapan program aplikasi NCI Bookman ver.3.10 pada perpustakaan PT.Bank,X(persero) Tbk | Abdul Khoir | 2005 | Skripsi ini membahas penerapan aplikasi NCI Bookman Ver.3.10 pada pengelolaan Perpustakaan PT. Bank "X" (Persero) Tbk. Skripsi ini menunjukkan bahwa aplikasi tersebut sudah memenuhi kebutuhan pengelolaan perpustakaan sehari-hari dan struktur meta data yang ada pada aplikasi sudah sesuai dengan kebutuhan perpustakaan. Akan tetapi, ada ketidaksesuaian dalam alur pengadaan pada aplikasi dan alur pengadaan perpustakaan, sehingga perpustakaan disarankan untuk menghubungi pengembang agar aplikasi versi seterusnya menyesuaikan dengan alur perpustakaan. Selain itu, dikarenakan terdapatnya beberapa bug dalam aplikasi, maka perawatan rutin disarankan untuk dilakukan. |
